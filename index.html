Lab Challenges & CTF Write-ups

OverTheWire – Bandit Level 15 → 16

Problem: Retrieve the password by talking to a localhost SSL service on port 30001.

Approach: Used openssl s_client -connect localhost:30001, sent the current password, received the next flag.

Tools: openssl, bash

Lesson: Certificate-less SSL sockets and the power of built-in *nix tools.

TryHackMe – Blue (EternalBlue)

Problem: Exploit SMBv1 vulnerability on Windows 7 to gain SYSTEM shell.

Approach: Nmap → Metasploit ms17_010_eternalblue → post-exploit hashdump & persistence.

Tools: nmap, Metasploit, Meterpreter

Lesson: Importance of patching and disabling legacy protocols.

HackTheBox – Starting-Point: Vaccine

Problem: Obtain user & root flags on a Linux box running a custom web app with SQLi & password reuse.

Approach: SQLMap → LFI → credential reuse → sudo enumeration → privilege escalation via path hijack.

Tools: SQLMap, Burp Suite, linPEAS

Lesson: Never reuse passwords across services; always check sudo version & PATH.
